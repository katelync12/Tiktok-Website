AP English Literature

Review Notes

This is a very comprehensive digital handbook of everything you need to know for AP English Literature. There are both rhetorical terms and sample essays.

Link to the notes: https://www.apstudynotes.org/english/

Cram Chart

This is a cram chart with the main concepts from English Literature.

Link to the cram chart: https://drive.google.com/file/d/1oMgEfS8HU3T6QcEhy5CLPxtsUX02Mcwe/view?usp=sharing

Review Videos

Collegeboard provides review videos for this course. These videos are quite long but thorough. They’ll be helpful if you have time.

Link to the playlist: https://www.youtube.com/playlist?list=PLoGgviqq4845L7Yj9c1kkIfdskkaLOUzD

Collegeboard Past FRQs

AP tests have similar formats every year. I’ve found that if you do enough practice tests, the same types of questions will begin to pop up. Collegeboard provides FRQs from the past years, scoring guidelines, and sample student responses.

Link to Past FRQs: https://apcentral.collegeboard.org/courses/ap-english-literature-and-composition/exam/past-exam-questions

Full Released Exams

Alongside FRQs, it’s also helpful to practice the multiple chose (MC) section of the exam. While Collegeboard does not release the MCs every year, there are some full released exams containing MCs that you can practice on.

Link to past exams: https://drive.google.com/drive/folders/1dK_HbTQjPBRlrK6y-CxgYSCOlCsrmlO5?usp=sharing
