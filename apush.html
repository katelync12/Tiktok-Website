Review Videos

Both Heimler’s History and Crash Course provides comprehensible videos for each unit. You’ll be able to get the overarching events/themes of each unit through these videos. He also connects themes across time periods which is quite helpful for contextualization. I personally used Heimler’s History for AP World History and Crash Course for APUSH, and did really well in both classes.

Link to Heimler’s History: https://www.youtube.com/c/SteveHeimler/playlists?view=50&sort=dd&shelf_id=3

Link to Crashcourse: https://www.youtube.com/playlist?list=PL8dPuuaLjXtMwmepBjTSG593eG7ObzO7s

Collegeboard Past FRQs

AP tests have similar formats every year. I’ve found that if you do enough practice tests, the same types of questions will begin to pop up. Collegeboard provides FRQs from the past years, scoring guidelines, and sample student responses.

Link to Past FRQs: https://apcentral.collegeboard.org/courses/ap-united-states-history/exam/past-exam-questions

Full Released Exams

Alongside FRQs, it’s also helpful to practice the multiple chose (MC) section of the exam. While Collegeboard does not release the MCs every year, there are some full released exams containing MCs that you can practice on.

Link to full exams: https://drive.google.com/drive/folders/1KyANT7TrBhGo5e7Yv0y-Mu24hWMtfUMr?usp=sharing
